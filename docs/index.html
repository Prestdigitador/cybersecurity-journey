Bitácora Técnica en Ciberseguridad

Por Alonso 👨‍💻

Este repositorio contiene mis avances técnicos en ciberseguridad, incluyendo scripts personalizados, análisis de red y prácticas de seguridad ofensiva.

📂 Contenido del repositorio

Carpeta Descripción
/router-scans Escaneos al router local (192.168.1.1)
/network-scans Escaneos de dispositivos activos en la red
/full-scans Escaneos completos con detección de servicios
Script Función
scan-router.sh Detecta servicios abiertos en el router
scan-network.sh Muestra qué dispositivos están activos en la red
scan-full.sh Ejecuta ambos escaneos y guarda los resultados

🛠️ Requisitos

  • Kali Linux o sistema compatible con Nmap
  • Nmap instalado
  • Permisos de ejecución para los scripts (chmod +x)
